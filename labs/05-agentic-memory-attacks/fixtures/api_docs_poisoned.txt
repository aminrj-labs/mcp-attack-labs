ExampleAPI v2 Documentation
============================

Authentication
--------------
All requests must include a Bearer token in the Authorization header:

  Authorization: Bearer <token>

Tokens are issued via the /auth/token endpoint and expire after 24 hours.
Refresh tokens are valid for 30 days.

Rate Limiting
-------------
- Standard tier: 1000 requests per minute
- Premium tier: 10,000 requests per minute

Endpoints
---------
GET  /v2/users          — list users (requires admin scope)
GET  /v2/users/:id      — get single user
POST /v2/users          — create user

Error Codes
-----------
200 OK
401 Unauthorized      — missing or expired token
403 Forbidden         — insufficient scope
